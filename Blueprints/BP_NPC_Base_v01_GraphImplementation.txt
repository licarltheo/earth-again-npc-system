# BP_NPC_Base_v01 - Blueprint Graph Implementation
# Copy this into Unreal Engine Blueprint Editor

## Event Graph

### BeginPlay Event
```
Event BeginPlay
├── Set CurrentState = Idle
├── Set LastSpawnTime = Now
├── Set HomeLocation = GetActorLocation()
├── Set HomeRotation = GetActorRotation()
├── Set InteractionCooldown = 30.0
├── Set Reputation = 0.5
├── Set bIsInteractable = true
├── Set bIsInDanger = false
├── Set Aggressor = None
├── Set NPCName = "NPC"
├── Set NPCRole = Citizen
├── Set NPCDistrict = NewAccra
├── Set IsKeyNPC = false
├── Set IsPersistent = true
├── Set CurrentState = Idle
├── Call DialogueComponent.InitializeDialogue(NPCName, DialogueIDs)
├── Call QuestComponent.InitializeQuests(QuestIDs)
└── Call BehaviorTreeComponent.StartTree(BehaviorTree)
```

### Tick Event
```
Event Tick(DeltaSeconds)
├── Call BehaviorTreeComponent.Update()
├── Call DialogueComponent.Tick(DeltaSeconds)
├── Call QuestComponent.Tick(DeltaSeconds)
├── Call UpdateStateMachine(DeltaSeconds)
└── If InteractionCooldown > 0.0
    └── Subtract DeltaSeconds from InteractionCooldown
```

### OnConstruction Event
```
Event OnConstruction(Transform)
├── Call UpdateAppearance()
├── Set HomeLocation = GetActorLocation()
├── Set HomeRotation = GetActorRotation()
├── Call DialogueComponent.InitializeDialogue(NPCName, DialogueIDs)
└── Call BehaviorTreeComponent.StartTree(BehaviorTree)
```

### Destroyed Event
```
Event Destroyed
├── If IsPersistent
│   └── Call UnregisterPersistentNPC()
├── Call QuestComponent.HideQuestMarker()
└── Call DialogueComponent.EndDialogue()
```

### IsAvailableForInteraction() -> Boolean
```
Function IsAvailableForInteraction() -> Boolean
├── If CurrentState == Working or CurrentState == Sleeping or CurrentState == Combat or CurrentState == Fleeing
│   └── Return false
├── If InteractionCooldown > 0.0
│   └── Return false
├── If bIsInteractable == false
│   └── Return false
└── Return true
```

### OnPlayerProximity(Player)
```
Function OnPlayerProximity(Player)
├── Calculate Distance = GetActorLocation() - Player.GetActorLocation()
├── If Distance < 500.0
│   ├── If InteractionCooldown > 0.0
│   │   └── Return
│   ├── If IsAvailableForInteraction() == false
│   │   └── Return
│   ├── Call DialogueComponent.StartDialogue(Player)
│   ├── Call PlayGreetingAnimation()
│   └── Set InteractionCooldown = 30.0
└── Return
```

### OnPlayerFar()

Function OnPlayerFar()

### OnDamage(DamageAmount, DamageCauser)
```
Function OnDamage(DamageAmount, DamageCauser)

### OnDeath()

└── Call DialogueComponent.EndDialogue()
```

### OnQuestComplete(QuestID)
```
Function OnQuestComplete(QuestID)
├── Call AddReputation(0.5)
└── Call QuestComponent.OnQuestComplete(QuestID)
```

### GetReputation() -> Float
```
Function GetReputation() -> Float
└── Return Reputation
```

### AddReputation(Amount)
```
Function AddReputation(Amount)
├── Set Reputation = Reputation + Amount
├── If Reputation > 1.0
│   └── Set Reputation = 1.0
├── If Reputation < 0.0
│   └── Set Reputation = 0.0
└── Call QuestComponent.OnReputationChange(Reputation)
```

### RemoveReputation(Amount)
```
Function RemoveReputation(Amount)
├── Set Reputation = Reputation - Amount
├── If Reputation > 1.0
│   └── Set Reputation = 1.0
├── If Reputation < 0.0
│   └── Set Reputation = 0.0
└── Call QuestComponent.OnReputationChange(Reputation)
```

### UpdateStateMachine(DeltaSeconds)
```
Function UpdateStateMachine(DeltaSeconds)
├── If CurrentState == Idle
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_IdleHandler)
├── If CurrentState == Walking
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_RoutineHandler)
├── If CurrentState == Working
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_RoutineHandler)
├── If CurrentState == Sleeping
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_RoutineHandler)
├── If CurrentState == Dialogue
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_InteractionHandler)
├── If CurrentState == Combat
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_DangerHandler)
├── If CurrentState == Fleeing
│   └── Call BehaviorTreeComponent.SelectSubtree(Subtree_DangerHandler)
└── If CurrentState == Questing
    └── Call BehaviorTreeComponent.SelectSubtree(Subtree_InteractionHandler)
```
